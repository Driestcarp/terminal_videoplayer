# A videoplayer for the terminal

### To compile:
```console
make term_vid
```

### Flaggs
<b> -loop </b> loops the video.

<b> -fps <i>xxx</i> </b> sets max fps to '<i>xxx</i>', '-1' for uncapped.

<b> -stats </b> shows stats of videoplayer.

<b> -size \<cols> \<rows> </b> sets terminal window to size of \<cols> and \<rows>.
